Generate new samples with diffusion models.
This is the code I used for my Barrett Honors Thesis (titled *Diffusion Models for Generating Samples in Imbalanced Datasets*) at Arizona State University during my final year of undergrad.
Perform training/generating all through the main_network.py script.
All results from training and generation will be saved in a new directory named "results_ML".
